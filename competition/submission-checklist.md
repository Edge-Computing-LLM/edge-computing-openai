# Devpost Submission Checklist

## Repository

- [ ] README explains one concrete problem and solution.
- [ ] Existing work and Build Week work are distinguished.
- [ ] All repository links work.
- [ ] No credentials are committed.
- [ ] No GGUF models are committed.
- [ ] No generated result files are committed.
- [ ] Codex Agent Skill is present.
- [ ] License is present.
- [ ] GitHub repository is public.

## Verification

- [ ] k3s node is Ready.
- [ ] NVIDIA GPU is visible.
- [ ] Kubernetes reports one allocatable GPU.
- [ ] NVIDIA RuntimeClass exists.
- [ ] Ollama, Open WebUI, OpenTelemetry Collector, Prometheus, and Grafana are Running.
- [ ] `edge-openai prove` reports `VERIFIED`.

## Video

- [ ] Video is under three minutes.
- [ ] Audio is understandable.
- [ ] Constrained hardware is shown.
- [ ] GPT-5.6 and Codex usage is explained accurately.
- [ ] Local inference and observability are shown.
- [ ] No private information is visible.

## Devpost

- [ ] Developer Tools category is selected.
- [ ] Public GitHub repository and video links are added.
- [ ] Screenshots are readable.
- [ ] Project description and competition terms are reviewed.
- [ ] Final submission is completed.
